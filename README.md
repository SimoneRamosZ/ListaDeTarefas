## 🔍 Sobre o Projeto

Este projeto é um **gerenciador de tarefas (To-Do List)** desenvolvido em C# com .NET, ideal para organizar, adicionar, remover e gerenciar tarefas.

Ele foi criado com foco em:

* 🧠 **Princípios de Clean Code**
* 🧩 **Organização por camadas (Controllers, Services, Models, Validators)**
* 📚 **Código legível e de fácil manutenção**

---

## 🚀 Funcionalidades

✔ Adicionar novas tarefas
✔ Marcar tarefas como completas
✔ Visualizar lista de tarefas ordenadas
✔ Estrutura modular com separação de responsabilidades

> *Possibilidades futuras:* filtros por status, API com frontend completo.

---

## 🧪 Tecnologias Utilizadas

✔ **C#**
✔ **.NET 10**
✔ Arquitetura em camadas
✔ Validações de dados (FluentValidation)
✔ Banco de dados SQLite
✔ Autenticação (ASP.NET Core Identity)

---

## 📦 Como Executar o Projeto

1. **Clone o repositório**

   ```bash
   git clone https://github.com/SimoneRamosZ/ListaDeTarefas.git
   ```

2. **Abra no Visual Studio ou editor de sua preferência**

3. **Restaure e execute**

   ```bash
   dotnet restore
   dotnet run
   ```
